A propos de moi et plus

Aprés une quinzaine d'années passées dans le milieu de l'industrie pétroliére, je me suis lancé dans une reconversion vers le développement web aprés une formation à la Wild Code School. Je m'y suis spécialisé sur PHP et son framework Symfony.

Outre le développement web j'ai également pour passion les jeux vidéos et la musique classique. Je suis violoncelliste dans un orchestre symphonique dont je gére la bibliothéque de partitions.

Parmi mes soft-skills je citerai : Autonomie, persévérance, sens de la communication, le travail en équipe, la rigueur et le sens de l'organisation.
